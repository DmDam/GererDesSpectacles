# GererDesSpectacles
ESEO Java EE Project : Réservation Spectacle

Une société de réservation des spectacles a décidé de mettre en place :

- Une application web « GestionSpectacles » qui permet de gérer la réservation des
spectacles grâce à un ensemble des services web. Ces services web seront accessibles
aussi par les agences de tourisme.

- Une application web pour permettre à ses clients de réserver des spectacles par Internet.
Cette application utilisera les services web de l’application « GestionSpectacles » au
même niveau que n’importe quelle autre agence de voyage. La gestion des clients sera
gérée par cette application.
